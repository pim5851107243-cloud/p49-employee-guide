## วิธีตอบ Checklist นี้

นิวตอบกลับมาในรูปแบบไหนก็ได้ · ตัวอย่าง 3 แบบ:

**แบบ 1 · ตอบทีละข้อ**
```
A1. ใส่ชื่อจริง - คุณนพพร / คุณมานะ / คุณสรินยา / คุณอัญชลี
A2. เก็บไว้แบบนี้
A3. ซ่อนไปก่อน
...
```

**แบบ 2 · เลือกจุดสำคัญที่อยากแก้ตอนนี้**
```
ขอแก้แค่: A1, A3, A5, C1, C3, E1, E2, E3, G1, H2
ที่เหลือเอาไว้รอบหน้า
```

**แบบ 3 · ให้ผมเดินตามลำดับความสำคัญ**
```
เริ่มจากข้อมูลจริงก่อน (A1, C1-C5, E1-E4, H2-H6)
ที่เหลือรอ
```

**สรุป**: มีทั้งหมด ~50 จุด decision · แต่ส่วนใหญ่ (~30 จุด) คือ "ตรวจข้อมูลจริง" ไม่ใช่การแก้ดีไซน์ · ใช้เวลาอ่าน ~10 นาที ตอบกลับ ~20 นาที เดี๋ยวผมแก้ทั้งหมดให้ใน session เดียวกัน
